# Keyboard Tester for Android
A simple app to test your physical or virtual keyboard and check it's working or not.
# Installation
Coming Soon!
# How to Use
Press any key on your physical (if you connected) or text something on virtual keyboard, If virtual keyboard not showing press 'Test' and then start typing on your physical keyboard.
