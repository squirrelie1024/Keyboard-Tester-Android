# Keyboard Tester for Android
A simple, lightweight application to verify the functionality of both physical and virtual keyboards on Android devices.
# Installation
Coming Soon!
# How to Use
Press any key on your physical (if you connected) or type something on virtual keyboard, If virtual keyboard not showing press 'Test' and then start typing on your physical keyboard.
